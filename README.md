Um site elegante, leve e altamente personalizável projetado para criar homenagens especiais e emocionantes para mães ou irmãs, seja no Dia das Mães, aniversários ou datas comemorativas.

O projeto foi desenvolvido pensando na máxima acessibilidade: ele pode ser clonado e modificado diretamente no código ou configurado de forma 100% visual através de um painel de administração secreto integrado, permitindo o uso tanto local quanto via hospedagem web (como GitHub Pages).

---

## 🚀 Modos de Funcionamento

O projeto foi estruturado sob uma lógica flexível que atende a dois perfis de usuários:

### 1. Modo Online / Hospedado (Sem Código)

Se o site estiver publicado na web (ex: GitHub Pages, Vercel, Netlify), qualquer pessoa pode personalizar a homenagem sem abrir um editor de texto.

- Basta acessar o site e pressionar o atalho **`Ctrl + Shift + E`**.
- Um painel de administração oculto será aberto para customizar textos, fontes, cores e fotos.

### 2. Modo Local (Desenvolvedor)

Para quem deseja controle total do projeto, alteração de arquivos de áudio originais ou modificações estruturais profundas:

- Basta clonar o repositório e rodar o arquivo `index.html` diretamente no navegador.
- Permite edição nativa do HTML, inclusão física de arquivos de mídia e alteração de scripts.

---

## ⚙️ Funcionalidades do Painel Administrador

Ao acionar o comando **`Ctrl + Shift + E`**, o painel administrativo (`admin.js` + `admin.css`) é carregado, liberando as seguintes opções:

- **Textos e Fontes:** Altere o título principal, mensagens de carinho e a tipografia do site.
- **Gerenciamento de Fotos:** Substitua as imagens de exibição e edite as legendas de cada uma individualmente.
- **Customização Visual:** Modifique a paleta de cores (fundo, textos, botões) e o estilo geral da página.
- **Aba Dados (Backup & Restauração):** Permite exportar (baixar) o arquivo de configuração com todas as alterações feitas. Isso garante que, se você precisar atualizar o site ou mudar de navegador, bastará importar esse arquivo para restaurar toda a sua personalização.
